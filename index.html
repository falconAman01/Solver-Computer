Courses

ADCA stands for Advanced Diploma in Computer Applications. It is a one-year, job-oriented diploma course designed to provide comprehensive training in computer applications, including MS Office(Word, Powerpoint, Excel, Access), HTML, and internet applications.

Curriculum that includes hands-on projects

  • Introduction with Windows 10,7
  • Computer personalizing
  • Paint
  • Wordpad
  • Notepad
  • Ms Word
  • Ms Excel
  • Ms Powerpoint
  • Ms Access
  • Html 5
  • Basic Hardware & Networking
