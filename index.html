📚 School Computer Course (Classes 1-10)

Comprehensive computer education curriculum for students from class 1 to 10, covering basic to intermediate computer skills and concepts.

Topics Covered by Class Level:

  • Classes 1-2: Introduction to Computers, Mouse & Keyboard Basics, Basic Shapes & Colors in Paint
  • Classes 3-4: Word Processing Basics, Simple Drawings, Internet Safety Introduction
  • Classes 5-6: MS Office Basics (Word, Excel), Basic Programming Concepts, Multimedia
  • Classes 7-8: Advanced Office Applications, Basic HTML, Introduction to Programming Logic
  • Classes 9-10: Computer Science Fundamentals, Data Structures Basics, Project Work

💻 Computer Science (Class 11-12)

Comprehensive Computer Science curriculum following WBCHSE syllabus for higher secondary students, covering programming, data structures, databases, and modern computing concepts.

Class 11 Syllabus:

  • Computer Fundamentals: Introduction to Computers, Number Systems, Boolean Logic
  • Programming in C++: Basic Programming Concepts, Control Structures, Functions, Arrays
  • Data Structures: Arrays, Stacks, Queues, Linked Lists
  • Database Management: SQL, Relational Database Concepts
  • Computer Networks: Network Fundamentals, Internet Concepts
  • Practical Programming: C++ Program Development and Debugging

Class 12 Syllabus:

  • Advanced Programming: Object-Oriented Programming, Inheritance, Polymorphism
  • Advanced Data Structures: Trees, Graphs, Searching and Sorting Algorithms
  • Database Systems: Advanced SQL, Database Design, Normalization
  • Operating Systems: Process Management, Memory Management, File Systems
  • Web Technology: HTML, CSS, JavaScript, Web Development
  • Software Engineering: SDLC, Software Testing, Project Management
  • Practical Work: Major Project Development, Viva Voce
